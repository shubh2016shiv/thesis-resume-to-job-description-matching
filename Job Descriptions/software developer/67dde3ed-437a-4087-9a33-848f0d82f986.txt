Title: Senior Java Software Developer Oxford Engineering Software
Full Description: A leading Engineering software company with a worldwide client base is seeking a talented Java developer to work in its Oxford team. You should be an accomplished Java developer and have previously worked in an engineering or scientific software background. An interest in architectural design, GUI design and efficient data handling would be an advantage.
